Homework 2021-02-03: Arduino Programming - Roller Blind
-------------------------------------------------------------------------------

Sie sollen eine einfache Ablaufsteuerung für ein Rollo implementieren (Hardware
und Software). 

Das State-Digramm ist gegeben. Folgende Events (Trigger) können auftreten: 
    UP (Taster)
    DOWN (Taster)
    STOP (Taster)
    BOTTOM_SWITCH (Taster)
    TOP_SWITCH (Taster)

Die Zustandsübergänge können folgende Aktivitäten ausführen:
    motor_right()       DC Motor läuft im Uhrzeigersinn
    motor_left()        DC Motor läuft gegen den Uhrzeigersinn  
    motor_stop()        DC Motor hält an 
    led_bottom_on()     LED_1 wird eingeschalten
    led_bottom_off()    LED_1 wird ausgeschalten
    led_top_on()        LED_2 wird eingeschalten
    led_top_off()       LED_3 wird ausgeschalten

A) Entwickeln Sie eine Schaltung (Arduino, LEDs, Taster, L293D, DC Motor) mit 
    der die Steuerung durchgeführt werden kann (die Endschalter werden manuell 
    betätigt).    
    PIN 3, 4, 5     L293 / DC Motor (DIR_A, DIR_B, ENABLE)
    PIN 2           LED_1
    PIN 6           LED_2 
    PIN 7           UP (Taster)
    PIN 8           DOWN (Taster)
    PIN 9           STOP (Taster)
    PIN 10          BOTTOM_SWITCH (Taster)
    PIN 11          TOP_SWITCH (Taster)
    Fügen Sie ein Foto der Schaltung zur Lösung hinzu.

B) Implementieren Sie das folgende Programm am Arduino
    i) Werten Sie die IO Ports mit einer Zykluszeit von 100ms aus 
    ii) Lesen Sie die 5 Taster-Zustände am begin der Zykluszeit ein 
        und speichern Sie in Variablen ab.
    iii) Implementieren Sie die Funktionen (siehe Aktivitäten) für die 
        Ansteuerungen der LEDs und Motoren. Dabei werden Variablen gesetzt, die
        am Beginn des nächsten Zyklus geschrieben werden.
    iv) Implementieren Sie die State Machine zur Steuerung des Rollos anhand des State Diagrams.         

C) Testen Sie Ihre Steuerung manuell:
    i) Start (Zustand OPEN): Motor stop, LED_1 ein
       [DOWN]: Motor Linkslauf, LED_1 aus 
       [BOTTOM_SWITCH]: Motor Stop, LED_2 ein

       [UP]: Motor Rechtslauf, LED_2 aus
       [TOP_SWITCH] Motor Stop, LED_1 ein 

    ii) Start (Zustand OPEN): Motor Stop, LED_1 ein
        [DOWN]: Motor Linkslauf, LED_1 aus 
        [STOP]: Motor Stop
        [DOWN]: Motor Linkslauf
        [BOTTOM_SWITCH]: Motor Stop, LED_2 ein

        [UP]: Motor Rechtslauf, LED_2 aus
        [STOP]: Motor Stop
        [UP]: Motor Rechtslauf
        [TOP_SWITCH] Motor Stop, LED_1 ein 

    iii) Start (Zustand OPEN): Motor stop, LED_1 ein
         [DOWN]: Motor Linkslauf, LED_1 aus 
         [STOP]: Motor Stop    
         [UP]: Motor Rechtslauf
         [TOP_SWITCH] Motor Stop, LED_1 ein 

    iv) Start (Zustand OPEN): Motor Stop, LED_1 ein
        [DOWN]: Motor Linkslauf, LED_1 aus 
        [BOTTOM_SWITCH]: Motor Stop, LED_2 ein

        [UP]: Motor Rechtslauf, LED_2 aus
        [STOP]: Motor Stop
        [DOWN]: Motor Linkslauf
        [BOTTOM_SWITCH]: Motor Stop, LED_2 ein

        [UP]: Motor Rechtslauf, LED_2 aus
        [TOP_SWITCH] Motor Stop, LED_1 ein
